type: project

2단계 청킹 파이프라인에서 각주(`[^N]`)를 본문 괄호 안에 인라인화할 것.

- 단순 교차참조 (예: `IAS 1의 문단 7 참조`) → `(IAS 1 문단 7 참조)`
- 실질 내용 (용어 정의, 기구 설명 등) → 괄호 안에 원문 그대로

**Why:** 청크 분할 시 `[^N]` 참조와 하단 각주 정의가 서로 다른 청크로 분리되면 의미 단절. 인라인하면 각 청크가 자기완결적(self-contained)이 됨.

**How to apply:** 1단계 구조 보존 마크다운은 건드리지 않고, 2단계 청킹 직전 전처리 단계로 각주 인라인화를 수행한다.
