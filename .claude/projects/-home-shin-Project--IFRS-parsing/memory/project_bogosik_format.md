docx 원본에서 표가 아닌 공백 정렬 텍스트로 된 재무제표 예시(예: K-IFRS 1001호 문단 102, 103)를 마크다운 표로 변환하는 기능.

패턴: 연속된 ContinuationText에서 `[항목명] [다중공백] [X/(X)/숫자]` 형태.

**Why:** 현재 파서는 이를 일반 텍스트로 처리하여 공백 정렬이 깨지고 가독성이 떨어짐. 표로 변환하면 구조가 명확해짐.

**How to apply:** 사용자가 `/보고식` 슬래시 명령어를 요청하면 그때 구현. 파서 레벨 자동 탐지가 아닌 명령어 기반 후처리로 구현할 것.
