NEXT_SESSION.md 파일을 업데이트하고 커밋하는 작업을 수행합니다.

## 절차

1. 이번 대화에서 수행한 작업을 분석합니다.
2. CLAUDE.md 파일을 읽고 현재 내용을 파악합니다.
3. "완료된 작업" 섹션에 이번 대화에서 완료한 작업을 추가합니다.
4. "다음 작업" 섹션을 필요에 따라 업데이트합니다 (완료된 항목 제거, 새로 발견된 작업 추가).
5. 변경된 CLAUDE.md를 저장합니다.
6. git add CLAUDE.md && git commit 으로 커밋합니다. 커밋 메시지 형식: "세션 저장: [주요 작업 요약]"

## 규칙

- 기존 완료 작업의 번호 체계를 이어서 매깁니다.
- 날짜는 오늘 날짜를 포함합니다.
- 핵심 파일 위치, 환경 등 기존 섹션은 필요한 경우만 업데이트합니다.
- 사용자에게 업데이트 내용 요약을 보여준 뒤 커밋합니다.
